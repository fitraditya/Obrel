# Obrel
Open source messaging system


## Features
* BYOS (bring your own server)
* Persistent messages
* Multiple channels
* Public and private channels
* Websocket API
* MIT licensed

## Installation
```
$ git clone https://github.com/fitraditya/Obrel.git
$ cd Obrel
$ npm install
$ npm install obrel-web
$ npm start
```
